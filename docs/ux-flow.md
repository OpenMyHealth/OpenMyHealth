# UX Flow (Safety-First)

## 1) Initialize Vault
- User sets passphrase once.
- Vault becomes encrypted local container.

## 2) Connect Source
- User starts source guide from side panel.
- Extension opens official source site.
- On-page copilot overlay shows step-by-step progress.

## 3) Capture Records
- User clicks capture from side panel.
- Source page is parsed locally.
- Records are normalized and stored in vault.

## 4) Ask Question
- User types health question in side panel.
- Local retrieval returns top candidate records.

## 5) Approval + Preview
- User checks/unchecks candidates.
- Preview shows exact context block to be sent.
- Sensitive tokens are redacted.

## 6) Inject Context
- Approved context is inserted into active AI chat input.
- Transfer event is logged in local transparency audit.
